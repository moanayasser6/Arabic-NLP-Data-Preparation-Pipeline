# Arabic-NLP-Data-Preparation-Pipeline
A Python-based preprocessing pipeline for Arabic Islamic fatwa datasets. The project standardizes data from multiple Excel sources, handles missing values, normalizes Arabic text, removes diacritics and duplicates, generates missing titles, and prepares a clean dataset for information retrieval and NLP applications.
## Features

- Multi-file Excel ingestion
- Column standardization
- Missing value handling
- Arabic text normalization
- Diacritics removal
- Duplicate detection and removal
- Automatic title generation
- Retrieval document creation

## Output

- clean_fatwas.xlsx
- clean_fatwas.csv

## Technologies

- Python
- Pandas
- OpenPyXL

## Final Dataset

117,809 cleaned fatwa records.
